### Development Setup

```bash
# Clone the repo
git clone https://github.com/lewiswigmore/frontmatter.git
cd frontmatter

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Open [localhost:3000](http://localhost:3000) to see the app.
